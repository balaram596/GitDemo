# College Management Portal

## Project Overview

The College Management Portal is a responsive web application developed using HTML, CSS, and Bootstrap 5. It provides a simple interface for managing student-related activities such as registration, attendance tracking, results management, student profiles, and contact information.

The project demonstrates the use of various Bootstrap components and responsive design principles.

---

## Features

### Home Page

* Responsive Navbar
* Hero Section
* About Section
* Services Cards
* Carousel for Announcements
* Collapse Component
* FAQ Accordion
* Footer with Social Media Links

### Student Registration

* Student Registration Form
* Form Validation
* File Upload
* Input Groups
* Required Field Validation

### Student Dashboard

* Statistics Cards

  * Total Students
  * Present Students
  * Absent Students
  * Placements
* Search Box
* Responsive Student Table
* Bootstrap Badges

### Student Profile

* Profile Card
* Student Information
* Profile Image

* Bootstrap Modal

### Contact Page

* Contact Form
* Alert Messages
* Responsive Layout

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Navigate through the pages using the navbar.
