# 🎓 SkillUp Training Institute Website

A responsive Training Institute web application developed using **HTML5, CSS3, Bootstrap 5, and JavaScript**. The website allows students to explore available courses, register online, learn about the institute, and contact the training center.

---

## 📌 Project Overview

SkillUp Training Institute Website is designed to provide an easy-to-use platform for students to:

* View available training courses
* Register for courses online
* Learn about the institute and faculty
* Contact the institute through a contact form
* Access course details and institute information

The project is fully responsive and built using Bootstrap components.

---

## 🚀 Features

### 🏠 Home Page

* Responsive Navigation Bar

2. Open the project folder.

3. Open `Home.html` in your web browser.

No additional installation is required.
